# PROMPT GUIDE（基础镜头精简版）

## 固定规则（必须）

1. **模型**：Gemini Banana Pro
2. **画幅**：**16:9**
3. **分辨率**：**1K（建议 1024×576）**
4. **固定场景**：夜晚旧公寓客厅兼书房，窗外有雨
5. **固定角色**：男A（深色夹克，30+）与女B（浅灰衬衫，28+）
6. **风格统一**：电影写实、胶片颗粒、暖钨丝灯 + 冷月光
7. **禁止项**：卡通、插画、夸张畸变、第三角色

---

## 镜头体系（仅保留）

- 景别：极特写、特写、近景、半身（中近景）、中景、全景、大全景
- 经典语法：正反打（shot-reverse-shot）
- 构图：三分法、居中对称、对角线、引导线、框中框、留白
- 视角：平视、仰视、俯视、监控视角（高位监控）

---

## 批量生成策略（务实）

- 串行执行（并发=1）
- 每次请求间隔 2~4 秒
- 遇 429 / 5xx：指数退避重试（2s → 4s → 8s → 16s）
- 文件命名：`public/shots/shot-001.png` 到 `shot-042.png`
