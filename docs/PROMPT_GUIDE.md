# PROMPT GUIDE（避免同质化）

## 目标
在**固定主场景 + 固定角色**前提下，通过「景别 / 构图 / 机位 / 运镜 / 光线 / 情绪」形成可感知差异，避免图片“像复制粘贴”。

- 主场景：夜晚旧公寓客厅兼书房，窗外有雨
- 固定角色：男A（深色夹克）/ 女B（浅灰衬衫）
- 统一风格：电影写实、胶片颗粒、暗色电影风

---

## 结构化模板（通用）

```text
场景：{空间描述 + 时间 + 天气 + 环境特征}
角色：{角色外观 + 状态 + 相对位置}
构图：{三分法 / 对称 / 前景遮挡 / 负空间 / 引导线...}
机位：{平视 / 俯拍 / 仰拍 + 方位角 + 焦段倾向}
运镜：{静止 / 推近 / 后拉 / 横移 / 跟拍 / 弧线环绕}
景别：{特写 / 近景 / 中景 / 全景 / 大全景}
光线：{主光、辅光、色温、对比度、是否逆光边缘光}
情绪：{压迫 / 冷静 / 暧昧 / 冲突 / 和解}
镜头语言限制：{禁止卡通、禁止插画、禁止第三角色、禁止夸张畸变}
```

---

## 分景别差异化模板

### 1) 特写（Close-up）
- 重点：面部细节、眼神、皮肤纹理、微表情
- 建议：85mm+，浅景深，背景极弱化

```text
景别：特写
机位：角色眼平或略低，85mm，浅景深
构图：脸部占画面 60-75%，仅保留少量前景线索
镜头语言限制：必须看清情绪细节；背景不可抢戏
```

### 2) 近景（Medium Close-up）
- 重点：上半身关系 + 手势动作
- 建议：50-85mm，保持人物与环境弱连接

```text
景别：近景
机位：平视 15°~35°，50-85mm
构图：上半身 + 前景遮挡，强调互动张力
镜头语言限制：动作细节可读，环境信息不超过 35%
```

### 3) 中景（Medium Shot）
- 重点：人物与环境关系平衡
- 建议：35-50mm，适合对白推进

```text
景别：中景
机位：轴线两侧反打，35-50mm
构图：人物占比 40-60%，保留环境上下文
镜头语言限制：需体现角色站位与空间关系
```

### 4) 全景（Wide Shot）
- 重点：空间建立、角色距离
- 建议：24-35mm，展示调度与动线

```text
景别：全景
机位：房间角落/门外，24-35mm
构图：完整展示场景骨架与人物位移
镜头语言限制：空间信息优先于情绪细节
```

### 5) 大全景（Extreme Wide）
- 重点：环境主导、人物渺小、命运感
- 建议：18-24mm，高位或远位机位

```text
景别：大全景
机位：高位俯视或远端透视，18-24mm
构图：人物占比 <20%，强调孤立与压迫
镜头语言限制：必须先读空间再读人物
```

---

## 反同质化策略

1. **一镜一变量强变化**：至少改变 2 个维度（如景别 + 机位）。
2. **焦段逻辑化**：特写用长焦，全/大全景用广角。
3. **情绪与光线绑定**：冲突提升对比度，和解降低反差。
4. **运镜语义明确**：推近=压迫/聚焦，后拉=疏离，横移=信息展开。
5. **强约束词**：加入“禁止卡通/插画/夸张畸变/第三角色”避免模型飘。

---

## 推荐生成流程（节流 + 重试）

- 并发：1（串行）
- 间隔：2~4 秒
- 重试：429 / 5xx 指数退避（2s, 4s, 8s, 16s）
- 文件映射：`shot-001.png` ↔ `shots[0]`

这样能保证质量一致性并降低接口限流风险。
